About Cows

Describing some concepts/terms in detail:

  • Document Object Model -
    • is constructed as a tree of objects and the browser creates a Document Object Model of the page when a web page is loaded.
  • innerHTML -
    • The innerHTML sets or returns the content of an element
  • Event listener -
    • This (add~) method attaches an event handler to an element without overwriting existing event handlers. Or (remove~) event handlers that have been attached with the addEventListener() method.
  • Function -
    • Declares a function
  • API -
    • is a medium to fetch or send data between interfaces.
  • AJAX
    • can read data from a web server - after the page has loaded, update a web page without reloading the page or send data to a web server - in the background.
